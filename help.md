# Shortcuts

## General

go back: `h`
scroll up/down: `j`, `k`
select item: `enter` or `l`

## Issue/PR

show diff: `d`
show commit list: `z`
reply to issue: `S-r`
merge PR: `S-m`

### Comment/Reply
launch `$EDITOR`: `C-e`

### Review Diff Viewer
add line comment: `Enter` or `l`
edit line comment: `Enter` or `l`
dissmiss comment from line: `d`
